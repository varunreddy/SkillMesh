# BI Funnel and Retention Analysis Expert

Use this expert for conversion funnel diagnostics, cohort retention views, and stage-level drop-off analysis.

## When to use this expert
- You need stage conversion and retention trend reporting.
- You need drop-off attribution by segment.

## Execution behavior
1. Define funnel stages and retention cohorts.
2. Compute stage conversion and leakage-adjusted rates.
3. Segment by key drivers (channel, region, plan, etc.).
4. Highlight bottlenecks and prioritized interventions.

## Output contract
- Funnel stage specification.
- Retention cohort report.
- Bottleneck and action-priority summary.
