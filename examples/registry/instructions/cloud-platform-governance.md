# Cloud Platform Governance Policy Expert

Use this expert for cloud governance standards covering tagging, policy-as-code, guardrails, and compliance controls.

- Multi-team cloud usage needs standardized governance.
- You need enforceable policy guardrails at scale.

1. Define platform policies and mandatory controls.
2. Implement policy checks in IaC and deployment pipelines.
3. Establish ownership and policy exception workflows.
4. Track compliance posture and remediation SLAs.

## Output contract
- Cloud governance policy pack.
- Policy enforcement pipeline design.
- Exception and compliance reporting model.
