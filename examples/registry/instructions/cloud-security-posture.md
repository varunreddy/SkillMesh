# Cloud Security Posture Management Expert

Use this expert for cloud security baseline hardening, misconfiguration detection, and posture monitoring.

## When to use this expert
- You need standardized cloud security controls.
- You need continuous visibility into misconfiguration risk.

## Execution behavior
1. Define baseline controls for identity, network, data, and compute.
2. Map controls to cloud-native security services.
3. Set continuous posture checks and remediation flows.
4. Track exceptions with approval and expiry policy.

## Output contract
- Cloud security baseline.
- Posture monitoring and remediation workflow.
- Exception governance register.
