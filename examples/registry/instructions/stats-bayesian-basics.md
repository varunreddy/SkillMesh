# Bayesian Modeling Basics Expert

Use this expert for prior/posterior reasoning, credible intervals, and Bayesian decision framing.

## When to use this expert
- You need probability statements about parameters.
- You need prior-informed modeling under limited sample sizes.

## Execution behavior
1. Define likelihood and prior assumptions transparently.
2. Fit posterior using appropriate inference approach.
3. Report posterior summaries and credible intervals.
4. Compare models or decisions using posterior quantities.

## Anti-patterns
- NEVER hide prior sensitivity when priors influence outcomes.
- NEVER mix frequentist and Bayesian intervals without clarification.

## Output contract
- Prior/likelihood specification.
- Posterior summary with credible intervals.
- Sensitivity notes and decision implications.
