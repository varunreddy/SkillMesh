# Frontend Data Visualization Integration Expert

Use this expert for integrating interactive charts into application UIs with reliable state and performance behavior.

## When to use this expert
- Product UIs embed analytics or chart-driven interactions.
- Chart state must synchronize with app state and routing.

## Execution behavior
1. Define chart interaction contract and state boundaries.
2. Optimize data fetch, transform, and render pipeline.
3. Ensure accessibility and responsive behavior.
4. Guard against expensive rerenders.

## Output contract
- Chart integration architecture.
- Interaction and state contract.
- Performance and accessibility checklist.
