# Frontend Performance Optimization Expert

Use this expert for rendering, bundle, and interaction performance optimization in web apps.

## When to use this expert
- Core Web Vitals or interaction latency is poor.
- Bundle size and runtime cost are growing.

## Execution behavior
1. Profile LCP/INP/CLS and hot rendering paths.
2. Optimize bundle splitting and dependency loading.
3. Reduce render churn with memoization and boundary design.
4. Validate improvements with before/after telemetry.

## Output contract
- Performance bottleneck report.
- Optimization plan with expected impact.
- Before/after metric comparison.
