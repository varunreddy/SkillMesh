# KPI Governance and Metric Contracts Expert

Use this expert for KPI definition governance, metric contracts, and cross-dashboard consistency controls.

## When to use this expert
- KPI definitions diverge across teams or tools.
- You need authoritative metric contracts with ownership.

## Execution behavior
1. Define KPI contract: formula, grain, filters, owner, and refresh policy.
2. Map upstream source dependencies and lineage.
3. Add validation tests for KPI parity across tools.
4. Define change-management and approval workflow.

## Output contract
- KPI contract registry.
- Lineage and ownership map.
- KPI parity test plan.
