# Frontend Testing and Quality Strategy Expert

Use this expert for frontend testing pyramids, stable UI tests, and release-quality guardrails.

## When to use this expert
- UI regressions are frequent.
- You need balanced unit/integration/e2e test coverage.

## Execution behavior
1. Define risk-based test matrix.
2. Add deterministic component and integration tests.
3. Limit e2e to critical user journeys.
4. Gate releases with quality thresholds.

## Output contract
- Frontend test strategy matrix.
- Critical-path e2e suite definition.
- CI quality-gate policy.
